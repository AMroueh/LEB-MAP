📡 Share & Deploy Your Map
This is a single self-contained HTML file. No server, no database — just one file that works anywhere.

Option 1 — Netlify Drop (fastest, 60 seconds, free)

1. Click "⬇ Export Standalone Map HTML" below to download the file
2. Go to app.netlify.com/drop
3. Drag the downloaded file onto the page
4. You get a live URL like https://abc123.netlify.app — that's your shareable link

Option 2 — GitHub Pages (permanent, free, your own domain)

1. Create free account at github.com → New repository
2. Name it lebanon-map → Upload the exported file, rename to index.html
3. Settings → Pages → Branch: main → Save
4. Your URL: https://YOUR-USERNAME.github.io/lebanon-map/

Option 3 — Embed in your website (iframe)

After hosting on Netlify or GitHub Pages, paste this into your site:

<iframe src="https://YOUR-URL-HERE.netlify.app" width="100%" height="720px" style="border:none; border-radius:6px;" title="Lebanon War Archive 2024–2026" loading="lazy"> </iframe>

Option 4 — Responsive embed (works on mobile)

<div style="position:relative;width:100%;padding-bottom:62%;height:0"> <iframe src="https://YOUR-URL-HERE.netlify.app" style="position:absolute;top:0;left:0;width:100%;height:100%;border:none" title="Lebanon War Archive" loading="lazy"> </iframe> </div>

Option 5 — WordPress

In any page/post, add an HTML block and paste the iframe code above.

Automated updates without engagement

To run updates without leaving a browser tab open, set up a GitHub Action that opens your Netlify URL every hour using a headless browser. This is more advanced but completely free. The system will auto-update your exported map each time it's opened in a browser with automation enabled.

Simplest approach: Keep the browser tab open. The 60-minute timer fires automatically — no clicks needed.
